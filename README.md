# Amazon-Sales-Analytics-Dashboard
An interactive Power BI dashboard developed to analyze Amazon sales performance across different markets, categories, countries, cities, and shipping modes. The dashboard transforms raw sales data into meaningful insights, enabling businesses to monitor performance and make data-driven decisions.

---

❗ Problem Statement

Amazon generates sales across multiple markets, categories, countries, cities, and shipping modes. Analyzing large volumes of sales data manually is time-consuming and makes it difficult to identify trends, monitor KPIs, and make informed business decisions. This project addresses these challenges by building an interactive Power BI dashboard that provides clear and actionable business insights.

---

🎯 Project Objectives

- Analyze overall sales performance.
- Monitor Total Sales, Profit, Quantity, and Shipping Cost.
- Compare sales across markets, categories, countries, and cities.
- Identify top-performing regions and shipping modes.
- Build an interactive dashboard for business decision-making.
